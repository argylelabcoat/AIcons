"desc/cl-64"
Color 64x64 icons

64x64 Color icons for use in window managers, doesn't seem to be much call for these any more.

Thanks to Detlef Schmier <detlef@mfr.dec.com> this section now uses the transparent color `None' whereever possible in the icons to allow better use of these icons by window managers.
