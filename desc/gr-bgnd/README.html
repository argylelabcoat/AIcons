"desc/gr-bgnd"
Grey Scale Tile Background Patterns

These icons are Grey Scale and as such often contain a large number of shades of grey in them. They are probably best used on a grey scale display terminal though many are also very suitable for color and monocrome displays too.

A M4 marco processed list of X Window commands to pick a background pattern from (usally via some `random line picker' program) is available in "support/xbgnd_cmds.m4". This file also lists some example color settings for bitmaps backgrounds and small scale patterns.

See Also:

desc/bw-bgnd Bitmap large scale background tiles.
desc/gr-bgnd Grey scale backgrounds
prog/patterns Small scale patterns.
