"desc/cl-32"
Color 32x32 Icons

This icon directory is huge, basically because 32x32 is the icon size used on both Macintosh and IBM PC clone computers. Most of these icons come from other collections many of which are listed in "docs/contribs".

Many of these icons, while they can not use transparency on PC computers, now have transparency added for the convenance of X Window Users.

Sub-Directories:

Due to the size of this collection, it has become imperative that the icons be split into smaller groups and sub-directories. The following are the a summery of those sub-directories.

Animals Animals and Creature Icons
Books Various books, which are open, closed, stacked, etc...

Buttons 3d A Collection of 3d buttons

CompEquip Computers, printers, modems, monitors, plugs...

Disks Computer Disks

Flags A Small collections of various flags from around the world. A much large collection can be found in "desc/cl-48/Flags".

Furniture beds, tables, chairs, lamps, bookcases, etc...

Games Icons for specific games.. chess, go, backgammon, tetris, war, golf...

Kites Icons of different types of kites

Network Network stuff, Servers, WWW, FTP, Gopher, Mail, News, Transfer,

Simpsons Basic Subset of icons from the Jeanette Foshee <x1foshee@exnet.iastate.edu> collection of simpson cartoon characters.

Toons Cartoon Characters
