"desc/cl-64x38"
Color icons in the CTwm icon size (64x38 pixels)

These icons are in the standard size as the icons supplied with the ctwm (Claude Tab Window Manager) X windows package. The original icons in this section were drawn by Claude Lecommandeur <Claude.Lecommandeur@epfl.ch> and Ion Cionca <Ion.Cionca@epfl.ch>. This section created when a large number of contributed icons of this size were made by Bill Morris <bmorris@borg24.jsc.nasa.gov> and some extra terminal icons by Dan Reynolds <dan@chpc.utexas.edu>.

Claude presumably looks like the icon "claude.xpm" in this directory, while the xmail mailbox icons are also duplicated in the "appl/mail" section of the library, and only included here for completeness.

Claude Lecommandeur has asked that the background (white) color of the icons in this directory be converted into the xpm `transparent' color. I have done so but am not eager to do it again.

NOTE: the original icons in the ctwm package used almost any color you care to think of. All these icons have been recolored to use the libraries standard color table (See docs/colortable).
