"appl/math"
Mathematical Symbols for Scientific Documents

This directory contains a set of mathematical symbols, super/subscrips and greek alphabet created by Karen Strom. These are normally converted into the GIF format and installed into a special WWW images directory "/kicons" for use within mathematical documents converted to HTML from LaTeX.

You can see their usage better in Anthony's WWW Images, where they have been converted to GIF image format.

You can download the latest copy from

      ftp://hanksville.phast.umass.edu/pub/misc/newgifs.tar.Z
Or example them in the directory
      http://donald.phast.umass.edu/kicons/

To convert the images back to gif format I suggest you use the "xbm2gif" script in the libraries "support/scripts" section. This script relies on one one other script found in the same directory and on the PbmPlus image filter package being somewhere in your path.

Example

   mkdir /www/images/Math
   cd /www/images/Math
   xbm2gif -t  AICONS/appl/math/*.xbm
