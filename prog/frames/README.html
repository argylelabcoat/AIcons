"prog/frames"
Framing templates you can use to `wrap' icons.

This directory contains numerous `templates' or borders you can use to surround the descriptive icons of various sizes, to create buttons.

Using the same template on a specific size and type of icons produces a collection which is coordinated. Use these as you will.

For example I can add a border around a 32x32 pixel description icon to create a button like those I use within the WWW images collection using the following ImageMagick command. 

      combine button.xpm -geometry +4+4 ../../desc/cl-32/castle.xpm \
              castle_button.gif
While I do not have a script to automatically add a template to a set of icons of a particular size, such a script should be easily created using the PbmPlus or other filter package.

Specific Templates

button 40 pixel template for use with 32 pixel icons
This last is the template I use for generating the "www/Buttons" from descriptive 32x32 icons.
butsmall 20 pixel template for use with 16 pixel icons
gfm 40 pixel template for use with 32 pixel icons
large 50 pixel template for use with 48 pixel icons
double double raised template
