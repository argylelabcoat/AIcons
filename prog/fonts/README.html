"prog/fonts"
Fonts Images for Text and Counters

These icons hold font set images which usually quite fancy and colorful, can not be used firectly in a black and white font file. That is to say that these letters are better used from a program or in generating larger pictures.

Many of the number fonts are extremely suitable to generating counters such as used on the www pages.

Specific Contribs:

"bubble_*.xpm" fonts
Bubble fonts using only 3 colors, and a white hightlight. The larger ones were from his program XMirrorMagic while the smaller ones are from "rocks n diamonds".

Anthony Thyssen (me!)
"drippy.xpm" was created from a raw black and white font found on the internet. The font was smoothed, displaced and overwritten on the original to produce the desired `dripping' font. The primary purpose was for a sign in my web castles dungeon (glowing moss on wall).

Numerous sets of Digits have suddenly appeared for web counters (odometers!) Some of these have been placed here as well. One collection of these counter digits is http://www.digits.com/charsets.html
