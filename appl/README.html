Application Icons. (sorted by application use)
Icons for use within specific applications or type of application.

An Examples listing is also available. WARNING: This displays lots images which are X bitmaps and X pixmaps which not all WWW browsers understand.

  • Cursors and associated masks. (IE: watch & gumby)
  • Icons for specific use in twm. (title bar buttons, ...)
  • Mail status indicator icons. (some with masks)
  • Icons for `logout' and exit buttons and symbols.
  • Icons for in drawing programs. (brushes, operations, ...)
  • Icons for in editing list and file data. (insert, append, ...)
  • Mathematical Symbols for use in WWW documents
  • Desktop Icons for file managers and directory listings.
    • Color Document icons.
    • Bitmap Document icons.
    • Color file index symbols for directory listings.
    • Bitmap file index symbols for directory listings.
    • Directories, Folders and Drawer icons.
    • Application type icons. (general)
    • 3D application and File Icon Set
