"prog/map/crossfire-bw"
map and landscape tiles (crossfire bitmaps)

Most of the icons are from 24x24 tiles used for the network UNIX game Crossfire. Some of these however have been colored by myself and are now used thoughout the world on the world wide web. A fully colored set of pixmaps are also available.
