"appl/desktop/file-cl"
File Symbols for Directory Listings (color)

This directory contains a set of color file symbols which can be used by applications for directory listings either directly or programmed in.

These symbols can be used by the Apatche and the NCSA httpd (WWW server). Look at the support file "HTTPD.srm.conf" in this directory.

It is suggested that any such symbol set be converted to GIF format before use however as most browsers (like netscape) do not understand the X pixmap format. (See the xbm2gif script)
