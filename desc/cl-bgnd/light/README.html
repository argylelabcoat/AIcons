"desc/cl-bgnd/light"
Light Colored Backgrounds

Light Colored Backgrounds suitable for text documents using a dark colored font. For example paper and wood textures.

Specific Contribs:

Thomas J. Albert http://www.city-net.com/albertfp/tja.htm
Scand the "wood_tan" image from some real wood for use on the Albert Forest products Home page http://www.city-net.com/albertfp/

"wood.xpm" was provided as a background pattern for Computer GO boards which traditionally are made from a light colored wood.
