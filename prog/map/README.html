"prog/map"
map and landscape tiles

In the sub-directories are found various tiles used to create maps and landscapes in plan view adventures or many sorts. They are presented here as a source for descriptive icons to create and use.

I also recomend you look at other sections of this library for buildings, objects, and walls to add other elements to the base landscapes created by these map tiles.

See Also:

prog/build small and large building for within towns
prog/walls walls, fences, and DIY mazes
prog/walls/castle_town castle and town walls
