"appl/desktop/file-bw"
File Symbols for Directory Listings (Bitmaps)

This directory contains a set of bitmap file symbols which can be used by applications for directory listings either directly or programmed in. My own program called "xbmbrowser" for example uses some of the Bitmap symbols to display non-image files it finds.

All the symbols in this directory are 20x20 pixels in size making them ideal for file listings and menus.

These symbols can be used directly by the NCSA httpd (WWW server). In fact this directory was started by this symbol set. Look at the support file "HTTPD.srm.conf" on the configuration options which needs to be set. It is suggested that any such symbol set be converted to GIF format before use however though most browsers do understand the X Bitmap.
