"desc/cl-misc"
Color icons of miscellanous size

Miscellaneous sized X pixmaps which do not easilly fit into the specific sized sections.

Thanks to Detlef Schmier <detlef@mfr.dec.com> this section now uses the transparent color `None' whereever possible in the icons to allow better use of these icons by window managers.

The special icons "colortable.xpm" and "colormap.xpm" contain the standard color table used by this library for your use and presual.
