"desc/bw-bgnd"
Large Scale Black and White Background Patterns

These bitmaps are large scale background tiles. These are generally used for generating root window background patterns which are different and interesting, allowing personalization of your display.

These patterns can also be used on color displays by specifing appropriate forground and background colors in the `setroot' command. By careful choice of colors some very weird effects can be produced.

A C-Preprocessed list of X Window commands to pick a background pattern from (usally via some `random line picker' program) is available in "support/xbgnd_cmds.m4". This file also lists some example color settings for the bitmap patterns in this directory as well as for the small scale bitmap patterns.

FYI: Spotty.xbm is my default black and white display backbround.
