"appl/edit"
A set of icons for editing list and file data 

  XmList Widgets
    append.xpm          - append new item to list
    change.xpm          - change existing item in list
    insert.xpm          - insert new item in list
    delete.xpm          - delete one item from list
    delete_grp.xpm      - delete a group of items in list
    delete_all.xpm      - delete all items in list

  Controling Commands
    build.xpm           - execute a command
    comments.xpm        - add comments to data
    info.xpm            - more detail information for command
    file.xpm            - save/load a file
    file_copy.xpm       - create a copy of the current file
    file_new.xpm        - create an empty file
    resize.xpm          - change size of data
    chart.xpm           - show statistics for data
    display.xpm         - display data
    exit.xpm            - exit application
