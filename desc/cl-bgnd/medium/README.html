"desc/cl-bgnd/medium"
Medium Colored Backgrounds

Medium colored Backgrounds (Grey?) Backgrounds suitable for both a very light and very dark font at the same time, usally Black and White. A specific color is often used for the image so a contracting text color can then be used. EG: denim is blue allowing a red text to be used without problems.
