"prog/bits-cl"
Color Bits and Pieces

The color pixmaps in this directory are NOT complete icons in themselves. They are small symbols, images and parts, which can be used to generate larger icons, symbols and displays, by mixing and matching.

Some conversion to use of transparency color performed by Detlef Schmier
