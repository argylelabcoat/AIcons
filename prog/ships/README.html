"prog/spaceships"
Spaceship pointing in 16 different directions

Various space ships pointing is different directions for use in battle games and other programs. The direction is 0-f (hexadecimal) starting straight up and rotating clock-wise.

A New ship design can easily be generated by creating a `0' ship pointing straight up and using the "xbm-rotate16" command in the "support/scripts" area of this library. This program works with pbmplus filters and will handle both Bitmaps and Pixmaps.
