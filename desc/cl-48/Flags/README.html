"desc/cl-48/Flags"
Color 48x48 Flags of the World

All the icons in this section are reproduced here curtsey of Steve Kinzler <kinzler@cs.indiana.edu> from his new "PIcons" collection which now handles the X pixmap format.

All the flags use the libraries standard color table with the only consession being the use of the X pixmap transparent color `None'.

NOTE: The file names in this section is the country identifiers used on the internet. For example Australian Flag can be found in the file "au.xpm".

The most recent version of this collection is downloadable from
http://www.cs.indiana.edu/picons/ftp/index.html
ftp://ftp.cs.indiana.edu/pub/faces/picons
This library includes hundreds of logos for Internet domains and Usenet newsgroups and thousands of user faces.
