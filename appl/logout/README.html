"appl/logout"
Icons used for Logout and Exit Buttons.

This Images can be placed on the display so that when the user selects it the user will either quit the current appliaction or logout of the login X window session.

For example: look at the `xlogout' script in the "support/scripts" directory of this library for an example use.

The Creation of these icons was thrown out as a challenge to the users on the xpm-talk mailing list <xpm-talk@babar.inria.fr> to start this library section off.
