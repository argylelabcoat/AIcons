"appl/desktop"
Icons for file managers and browsers

This icons are for the use of desktop programs, like file managers, directory listings, etc... They are any and all icons to do with files, documents, folders and directories.

The icons are futher grouped into the following sub-sections

docs-cl Color Document icons.
docs-cl48 Color Document icons. (48x48)
docs-bw Bitmap Document icons.
file-cl Color file index symbols for directory listings.
file-bw Bitmap file index symbols for directory listings.
dirs Directories, Folders and Drawer icons.
appl Application type icons. (general)
3d 3D application and File Icon Set
kde_32 KDE 32x32 Icon Set
kde_48 KDE 48x48 Icon Set

Specific application icons can be found in the appropriate descriptive icons sections.
