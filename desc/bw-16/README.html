"desc/bw-16"
Black & White 16x16 (button) Bitmaps

These icons are basically tiny bitmap icons which are commonly used for small control panels, menu selction indicators, toggle buttons, etc... In other words just a collection of very small symbols.

A number of buttons have on and off selections with and without highlight. They are designed for some sort of `radio_button' control panel or for menu selections. All of these have the following naming convention:-

         *_off.xbm       button off
         *_off_hlt.xbm   button is off but highlighted
         *_on.xbm        button on or selected
         *_on_hlt.xbm    button is on and is also hightlighted
These icons are currently:- check, tickbox, panel, raised, and a set of radio buttons and toggles in different sizes.

The z_buttons* files while not 16x16 in size is included as a `wrapper' for the icons in this directory.
